## 🛠️ The Avionics Stack Behind This Site

This platform is a fully custom-modeled responsive cyberspace ecosystem designed to represent technical excellence:
- **Client System:** React 19 + TypeScript bundled through Vite.
- **Micro-Animations:** Fluid state motion layers synchronized via `motion` (Framer Motion).
- **Interactive Cyber-Effects:** Real-time canvas rendering of electrical neuron gridlines (`CyberpunkParticles`).
- **Cloud Database:** Real-time synchronizations, adaptive local cache handling, and automatic multi-tab sync powered by **Google Cloud Firestore**.
- **AV deck configuration:** Interactive control panel allowing real-time switching of soundtracks, theme profiles, neon accents, and custom background loops.
